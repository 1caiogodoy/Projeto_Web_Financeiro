# 💰 Sistema de Controle Financeiro

Projeto desenvolvido para gerenciamento de finanças pessoais, permitindo visualizar a organização dos gastos através de gráficos.

## 🚀 Funcionalidades

- Inserção de dados financeiros (renda, gastos, investimento, lazer e dívidas)
- Geração de gráficos automáticos com Python (Matplotlib)
- Sugestão de organização financeira
- Integração entre Frontend e Backend
- Armazenamento de dados com SQLite
- Atualização dinâmica com JavaScript

## 🛠️ Tecnologias utilizadas

- Python (Flask)
- HTML, CSS, JavaScript
- Matplotlib
- SQLite

## 📊 Como funciona

1. O usuário preenche o formulário
2. Os dados são enviados via JavaScript (fetch)
3. O Flask processa os dados
4. Os gráficos são gerados dinamicamente
5. Os resultados aparecem na tela sem reload

## ▶️ Como rodar o projeto

# Clone o repositório
git clone https://github.com/seuuser/site-financeiro.git

# Entre na pasta
cd site-financeiro

# Crie o ambiente virtual
python -m venv venv

# Ative o ambiente
# Windows:
venv\Scripts\activate
# Linux/Mac:
source venv/bin/activate

# Instale as dependências
pip install flask matplotlib

# Execute o projeto
python main.py

# Acesse no navegador:
http://127.0.0.1:5000
